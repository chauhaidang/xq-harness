# Documentation Map

This directory holds contributor documentation, product notes, story records,
and architecture decisions for the repository.

## Main Files

- `ARCHITECTURE.md`: architecture discovery and boundary rules.
- `modules/ios-react-native-shell-adoption.md`: consumer guide for adopting the
  native iOS shell plus React Native remote payload pattern.

## Folders

- `learning/`: hands-on learning notes and step-by-step guides.
- `modules/`: polyglot module registry docs; start at
  [`modules/onboarding.md`](modules/onboarding.md) when migrating an external repo.
- `product/`: current product notes.
- `stories/`: feature notes and backlog.
- `decisions/`: durable decisions and tradeoffs.
